### 병합 정렬

- 말 그대로 범위를 나누고 합칠 때 정렬하는 알고리즘
- 분할 정복 알고리즘 중의 하나

---

### 병합 정렬의 구체적인 구현 단계

1. 분할: 입력받은 배열을 두 개의 똑같은 길이를 가진 부분 배열로 나눈다.
2. 정복: 부분 배열을 정렬한다. 부분 배열의 길이가 하나가 아니라면 재귀로 다시 계속 분할 시킨다.
3. 결합: 정렬된 모든 부분 배열을 하나의 배열로 만든다.

### 합병 정렬(Merge sort)의 코드

```python
def merge_sort(low, high):
    if high - low < 2:
        return
    mid = (low + high) // 2

		# 1. 분할: 앞쪽의 부분 배열과 뒤쪽의 부분 배열을 분할
    merge_sort(low, mid)
    merge_sort(mid,high)

    temp = []
    l, h = low, mid

		# 2. 정복: 부분 배열들을 정렬
    while l < mid and h < high:
				# 오름차순으로 정렬
        if arr[l] <= arr[h]:
            temp.append(arr[l])
            l += 1
        else:
            temp.append(arr[h])
            h += 1
    while l < mid:
        temp.append(arr[l])
        l += 1
    while h < high:
        temp.append(arr[h])
        h += 1

		# 3. 결합: 정렬된 값들로 기존 배열의 값들을 바꿔준다
    for i in range(low, high):
        arr[i] = temp[i - low]

N = int(input())
arr = list(map(int, input().split()))
merge_sort(0, N)
```
---
### 합병 정렬 시간 복잡도
$O(NlogN)$

- N은 합 칠때 N개의 데이터들이 존재하기 때문이다.
- logN은 N개의 데이터들을 나눌 때 호출레벨이다.
