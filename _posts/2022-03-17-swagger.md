## Swagger

개발한 REST API를 편리하게 문서화 해주고, 이를 통해서 관리 및 제 3의 사용자가 편리하게 API를 호출해보고 테스트 할 수 있는 프로젝트 입니다.

Spring Boot에서는 간단하게 springfox-boot-starter를 gradle dependencies 에 추가함으로 사용할 수 있습니다.

다만 주의할 점은 운영환경과 같은 외부에 노출되면 안되는 곳에는 사용할 때 주의해야 합니다.

## Swagger Annotation

![스크린샷 2022-02-24 오후 11.02.46.png](https://s3-us-west-2.amazonaws.com/secure.notion-static.com/eadabb17-bacf-4fc1-9606-bd25e7143290/스크린샷_2022-02-24_오후_11.02.46.png)

### Api

해당 어노테이션을 활용해 클래스를 스웨거에 표시할 수 있습니다.

```java
@Api(tags = {"API 정보를 제공하는 Controller"})
@RestController
@RequestMapping("/api")
public class ApiController {
...
}
```

![스크린샷 2022-02-26 오전 12.16.04.png](https://s3-us-west-2.amazonaws.com/secure.notion-static.com/cb15cd9a-d359-415d-83cc-b3d8cff6d202/스크린샷_2022-02-26_오전_12.16.04.png)

### ApiParam

파라미터의 이름을 지정해줍니다.

```java
@GetMapping("/plus/{x}")
public int plus(
		@ApiParam(value = "x값")
		@PathVariable int x,

		@ApiParam(value = "y값")
		@RequestParam int y){
	return x+y;
}
```

### ApiProperty

Request와 Response의 데이터들의 이름을 지정해줍니다.

```java
@ApiModelProperty(value = "사용자의 이름", example = "jihun", required = true)
private String name;

@ApiModelProperty(value = "사용자의 나이", example = "10", required = true)
private int age;
```

### ApiOperation

메소드 설명을 붙여주는 어노테이션입니다.

```java
@ApiOperation(value = "사용자의 이름과 나이를 리턴하는 메소드")
@GetMapping("/user")
public UserRes user(UserReq userReq){
		return new UserRes(userReq.getName(), userReq.getAge());
}
```

### ApiResponse

아래 http 통신 프로토콜의 조건을 설명해주는 어노테이션입니다.

```java
@ApiResponse(code = 502, message = "사용자의 나이가 10살 이하일때")
```

### ApiImplicitParam

`ApiParam` 어노테이션 대신에 해당 매소드 전체의 값들을 배열로 이름을 지정해주는 어노테이션입니다.

```java
@ApiImplicitParams(
            {
                    @ApiImplicitParam(value = "x값"),
                    @ApiImplicitParam(value = "y값")
            }
    )
    @GetMapping("/plus/{x}")
    public int plus(
            @PathVariable int x,
            @RequestParam int y){
        return x+y;
    }
```
