## git add

git을 사용해 버전관리를 하겠다고 선언한 이후에 해당 디렉토리에서 파일을 추가하는 명령어


## git add 실습

1. **f1.txt 파일을 추가해준다.**
    
    ![](https://jihun-24k.github.io/assets/images/git/git_add1.png){: .align-center}
    
- vim 으로 f1.txt 파일을 추가
- cat 으로 f1.txt 파일의 내용 확인
- ls -al로 해당 파일이 잘 추가 되었는지 확인

2. **git add 하기 이전**

![](https://jihun-24k.github.io/assets/images/git/git_add2.png){: .align-center}

- git status로 현재 git의 상태를 확인한다.
- f1.txt 파일은 현재 git으로 지정된 디렉토리 안에 있지만 git으로 관리하지는 않은 상태이다.

3**. git add [파일명]**

![](https://jihun-24k.github.io/assets/images/git/git_add3.png){: .align-center}

- git add f1.txt을 해주면 git에게 f1.txt도 이제 버전 관리를 할 수 있게 되도록 한다.
- 다시 git status로 상태를 확인하면 추가된 것을 확인할 수 있습니다.
