### 문제

![](https://jihun-24k.github.io/assets/images/baek_img/trees1.png){: .align-center}

- 입력으로 Inorder과 Postorder가 주어진다.
- Preorder를 출력
    
---

### 문제 접근 방법

- 포스트 오더의 마지막은 root 노드인 것을 활용한다.
- 인오더와 포스트 오더의 인덱스를 나눠가며 root 노드를 찾아내간다.

---

### 풀이 코드

```python
# 백준 2263번 트리의 순회(골드)
import sys
sys.setrecursionlimit(int(1e9))

n = int(input())
in_order = list(map(int, input().split()))
post_order = list(map(int, input().split()))

pos = [0] * (n+1)
for i in range(n):
    pos[in_order[i]] = i

def pre_order(i_start, i_end, p_start, p_end):

    if (p_start > p_end) or (i_start > i_end):
        return

    root = post_order[p_end] 
    print(root, end = " ")
    
    left = pos[root] - i_start
    right = i_end - pos[root]

    pre_order(i_start, i_start + left -1 , p_start, p_start + left -1)
    pre_order(i_end-right+1, i_end, p_end-right, p_end-1)

pre_order(0,n-1,0,n-1)
```
