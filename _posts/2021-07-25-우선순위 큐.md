### 우선순위 큐

- 우선순위 큐는 **우선순위가 가장 높은 데이터를 가장 먼저 삭제하는 자료구조**이다.
- 우선순위 큐는 데이터를 우선순위에 따라 처리하고 싶을 때 사용한다.
    - 예시) 물건 데이터를 자료구조에 넣었다가 가치가 높은 물건부터 꺼내서 확인해야 하는 경우

| 자료구조 | 추출되는 데이터 |
|:-----:||:--------:|
| 스택 | | 가장 나중에 삽입된 데이터 |
| 큐 | | 가장 먼저 삽입된 데이터 |
| 우선순위 큐 | | 가장 우선순위가 높은 데이터 |


### 우선순위 큐는 구현하는 방법

1. 단순히 리스트를 이용하여 구현할 수 있다.
2. 힙(heap)을 이용하여 구현할 수 있다.

- 데이터의 개수가 N개일 때, 구현 방식에 따라서 시간 복잡도를 비교한 내용은 다음과 같다.

| 우선순위 큐 구현방식 | 삽입 시간 | | 삭제 시간 |
|:-----:||:--------:||:-----:|
| 리스트 | | O(1) | | O(N) |
| 힙 | | O(logN) | | O(logN) |

- 단순히 N개의 데이터를 힙에 넣었다가 모두 꺼내는 작업은 정렬과 동일하다.(힙정렬)
    - 이 경우 시간 복잡도는 <b>O(NlogN)</b>이다.

---

### 힙(Heap)의 특징

- 힙은 완전 이진 트리 자료구조의 일종이다.
- 힙에서는 **항상 루트노드(root node)를 제거**한다.
- **최소 힙(min heap)**
    - 루트 노드가 가장 작은 값을 가진다.
    - 따라서 값이 작은 데이터가 우선적으로 제거된다.
- **최대 힙(max heap)**
    - 루트 노드가 가장 큰 값을 가진다.
    - 따라서 값이 큰 데이터가 우선적으로 제거된다.

### 힙에서 원소를 삽입할 때

- 새로운 원소가 삽입되었을 때  <b>O(logN)</b>의 시간 복잡도로 힙 성질을 유지하도록 할 수 있다.

![](https://jihun-24k.github.io/assets/images/data_struct/prior3.png){: .align-center}

- 만일 2가 삽입됐다고 가정한다면 2는 부모 노드인 9와 비교해 바꾸고 그 다음 부모 노드랑 비교해 바꿔 2번만 비교해서 정렬할 수 있다.
- 이는 전체 N번만큼 비교하지 않아도 되기에 시간 단축을 할 수 있다.

---

### 힙에서 원소를 제거될 때

- 원소가 제거되었을 때 <b>O(logN)</b>의 시간 복잡도로 힙 성질을 유지하도록 할 수 있다.
    - 원소를 제거할 때는 가장 마지막 노드가 루트 노드의 위치에 오도록 한다.

### 우선순위 큐 라이브러리를 활용한 힙 정렬 구현 예제

```python
import sys
import heapq
input = sys.stdin.readline

def heapsort(iterable):
    h = []
    result = []
    # 모든 원소를 차례대로 힙에 삽입
    for value in iterable:
        heapq.heappush(h, value)
    # 힙에 삽입된 모든 원소를 차례대로 꺼내어 담기
    for i in range(len(h)):
        result.append(heapq.heappop(h))
    return result

n = int(input())
arr = []

for i in range(n):
    arr.append(int(input()))

res = heapsort(arr)

for i in range(n):
    print(res[i])
```

- Python의 경우 기본적으로 제공되는 heap 라이브러리는 min_heap이다.
- 만일 max_heap이 필요하다면 값을 삽입할 때랑 꺼낼 때 값 앞에 **- (마이너스)**를 붙여준다.

### 최대 힙 응용

- 힙(heap)에 튜플(tuple)을 원소로 추가하거나 삭제하면 튜플 내에서 맨 앞에 있는 값을 기준으로 최소 힙이 구성된다.
- 따라서 최대 힙을 만들려면 각 값에 대한 우선 순위가 구한 뒤, **(우선순위, 값)**의 구조를 갖는 튜플을 힙에 추가해주거나 삭제해준다.
- 그리고 힙에서 값을 읽어 올 때 각 튜플에서 인덱스 1에 있는 값을 사용한다.

```python
import heapq

nums = [4, 1, 7, 3, 8, 5]
heap = []

for num in nums:
  heapq.heappush(heap, (-num, num))  # (우선 순위, 값)

while heap:
  print(heapq.heappop(heap)[1])  # index 1
```
