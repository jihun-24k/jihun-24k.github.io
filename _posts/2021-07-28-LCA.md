### LCA 개념

- Lowest Common Ancestor의 약자로 한글로는 최소 공통 조상이다.
- 두 원소들의 가장 가까운 부모 노드를 찾는 알고리즘
- 대부분 세그먼트 트리 또는 DP로 접근해서 해결

---

### LCA 알고리즘 진행 과정 (DP)

- 모든 노드의 부모 노드 정보를 담기 위해 parent 배열 선언
- 모든 노드의 깊이 정보를 담고 있는 depth 배열 선언
- 두 노드 u,v 의 LCA 찾아내기
    1. depth[u] > depth[v] 일 때 u를 parent[u]로 대체하면서 깊이 차이 맞추기
    2. u ! = v 일 때, 같은 부모 노드가 나올 때까지 u = parent[u], v = parent[v]를 반복

---

### LCA 알고리즘 코드 구현

```python
import sys
sys.setrecursionlimit(int(1e5))
input = sys.stdin.readline

N = int(input())
parent = [0] * (N + 1)  # 부모 노드 정보
d = [0] * (N + 1)       # 각 노드까지의 깊이(depth)
c = [0] * (N + 1)       # 노드 깊이를 계산했는지 여부
graph = [[] for _ in range(N + 1)] # 그래프 정보

for _ in range(N - 1):
    a, b = map(int, input().split())
    graph[a].append(b)
    graph[b].append(a)

# 모든 노드의 깊이 정보를 dfs로 설정
def dfs(x, depth):
    c[x] = True
    d[x] = depth
    for y in graph[x]:
        # 이미 계산한 노드라면 skip
        if c[y]:
            continue
        parent[y] = x
        dfs(y, depth + 1)

# LCA 찾는 함수
def lca(a, b):
    # 1.두 노드의 깊이를 맞춘다
    while d[a] != d[b]:
        if d[a] > d[b]:
            a = parent[a]
        else:
            b = parent[b]
    # 2.두 노드의 같은 부모를 찾는다
    while a != b:
        a = parent[a]
        b = parent[b]
    return a

dfs(1, 0)

M = int(input())
for i in range(M):
    a, b = map(int, input().split())
    print(lca(a, b))
```
