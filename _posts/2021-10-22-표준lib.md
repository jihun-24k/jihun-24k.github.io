### 표준 라이브러리 활용

코딩을 하다보면 유용한 기능들을 활용하기 위해서 표준 라이브러리들을 쓰는 경우가 많습니다. 아래와 같이 4가지 대표적인 라이브러리, 패키지들을 알아보도록 하겠습니다.

---

### dart:core 패키지

core 패키지는 다트의 기본적인 내용을 다루고 있으며 별도로 패키지 사용을 명시하지 않아도 자동으로 추가되어 있습니다. 예를 들어 print()함수와 리스트, 집합, 맵의 자료구조등이 core 패키지에 정의되어 있습니다. 

core 패키지에서 가장 먼저 알아야 할 것은 형변환에 관한 내용입니다. 많이 사용되어지는 타입들인 숫자와 문자열을 상호 변환할 수 있어야 합니다. 많은 경우 GUI의 사용자 입력이나 네트워크를 통한 URL이 문자열로 요청되기 때문입니다. 

---

### dart:io 패키지

dart:io 패키지에서는 사용자에게 입력을 받거나 파일을 읽고 쓰는 등의 입출력(I/O) 동작을 제공합니다. 가장 기본이 되는 내용은 다음과 같습니다.
- 사용자 입력 받기
- 새로운 파일을 생성하고 파일에서 값을 읽기
- 파일에 쓰기

io 패키지를 임포트하는 문장은 다음과 같습니다.
```dart
import 'dart:io';
```

사용자 입력을 받는 예제는 아래와 같습니다.
```dart
// 사용자 입력 받기
  stdout.write('Enter name? '); //print()와 같음
  String input = stdin.readLineSync();
print('Hello, $input');
```

파일 입출력을 받는 것은 아래와 같습니다.
```dart
import 'dart:io';

void main(){
  // 새로운 파일 생성하기
  File newfile = File('temp_file.txt');
  newfile.createSync();

  // 파일을 읽기(동기 방식)
  File poem = File('poem.txt');
  List<String> lines = poem.readAsLinesSync();
  for (String line in lines){
    print(line);
  }

  // 파일을 쓰기 (동기 방식)
  File memoFile = File('diary.txt');
  String contents = """
  2019.06.19: 플러터 공부 시작
  
  오늘은 플러터의 표준 라이브러리를 공부했습니다.
  안드로이드 스튜디오를 활용하여 예제를 실행했다.
  """;
  memoFile.writeAsBytesSync(contents);
}
```

### dart:math 피키지

math 패키지는 수학에 관한 내용을 다루는 패키지입니다. 기초적인 수학 함수들을 포함하고 있습니다.
- 최댓값 최솟값 -> min(), max()
- 제곱근 -> sqrt()
- 난수 발생 -> _makeRandomNumbers()

---

### dart:convert 패키지

convert 패키지는 모바일과 웹 프로젝트에서 빈번하게 사용되는 JSON 형식을 다루는 패키지입니다. 

- JSON 문자열 파싱
- JSON 파일 읽기 -> _readBasketJson()
- JSON 파일 쓰기 -> writeAsStringSync(jsonEncode())
- _readBasketJson() 파일을 다시 읽어오고 다음 갱신된 값이 제대로 저장됐는지 확인합니다.
