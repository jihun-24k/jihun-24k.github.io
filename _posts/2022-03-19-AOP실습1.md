## Aspect

해당 클래스가 AOP로 동작하게 하려면 붙여줘야하는 Annotation입니다. 

## PointCut

사용자가 어느 부분에 적용시킬 것인지 지정해주는 Annotation입니다. 수식이 굉장히 많기에 사용할 때마다 정확한 수식을 찾아보는게 좋습니다.

```java
@Pointcut("execution(* com.example.aop.controller..*.*(..))")
private void cut(){ }
```

## Before & AfterReturning

Before와 After는 지금 어떤 메소드가 입력되고 출력되는지 로그를 남기고 싶을 때 사용하는 Annotation입니다. 

```java
		@Before("cut()")
		public void before(JoinPoint joinPoint){
        MethodSignature methodSignature = (MethodSignature) joinPoint.getSignature();
        Method method = methodSignature.getMethod();
        System.out.println(method.getName());

        Object[] args = joinPoint.getArgs();
        for(Object obj : args){
            System.out.println("type : " + obj.getClass().getSimpleName());
            System.out.println("value : " + obj);
        }
    }

    @AfterReturning(value = "cut()", returning = "returnObj")
    public void afterReturn(JoinPoint joinPoint, Object returnObj){
        System.out.println("return obj");
        System.out.println(returnObj);
    }
```
