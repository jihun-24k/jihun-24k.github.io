## Adapter pattern

어뎁터는 실생활에서는 돼지코라고 불리는 변환기인데 이 역할은 프로그래밍 세계에서도 똑같다.

호환성이 없는 기존 클래스의 인터페이스를 변환하여 재사용 할 수 있도록 해준다.

SOILD 중에서 OCP를 따른다.


## Adapter 예제

예를 들어 110v 헤어드라이기와 220v 청소기가 있다고 가정해보자.

그렇다면 전자기기들을 규제해주는 110v와 220v는 2개의 인터페이스로 작성이 가능하고

각각의 객체들인 헤어드라이거와 청소기는 클래스로 작성이 가능해진다.

- 110v 인터페이스
    
    ```java
    package com.company.adapter;
    
    public interface Electronic110V {
        void powerOn();
    }
    ```
    
- 220v 인터페이스
    
    ```java
    package com.company.adapter;
    
    public interface Electronic220V {
        void connect();
    }
    ```
    
- 헤어드라이기 클래스
    
    ```java
    package com.company.adapter;
    
    public class HairDryer implements Electronic110V{
        @Override
        public void powerOn() {
            System.out.println("헤어 드라이기 110v on");
        }
    }
    ```
    
- 청소기 클래스
    
    ```java
    package com.company.adapter;
    
    public class Cleaner implements Electronic220V{
        @Override
        public void connect() {
            System.out.println("청소기 220v on");
        }
    }
    ```

- 메인 클래스에서 실행
    
    ```java
    package com.company;
    
    import com.company.adapter.Cleaner;
    import com.company.adapter.Electronic110V;
    import com.company.adapter.HairDryer;
    import com.company.adapter.SocketAdapter;
    
    public class Main {
    
        public static void main(String[] args) {
            HairDryer hairDryer = new HairDryer();
    				connect(hairDryer);
    
    				// 청소기는 220v라 호환이 되지 않는다.
            Cleaner cleaner = new Cleaner();
    				connect(cleaner);
        }
    
        public static void connect(Electronic110V electronic110V){
            electronic110V.powerOn();
        }
    }
    
    ```
    
    메인 클래스의 connect 메소드는 110v를 받아와 전원을 켜오는 기능을 지니고 있다.
    
    하지만 청소기 인스턴스를 생성해 connect를 시도해보면 호환이 되지 않는다.
    
    이유는 청소기는 220v로 연결이 되기 때문이다. 이러한 문제를 Adapter가 해결해준다.
    

### Adapter 클래스 생성

```java
package com.company.adapter;

// 1. 110v에서 220v를 변환하기 위해 먼저 110v를 상속받는다.
public class SocketAdapter implements Electronic110V{

		// 2. 220v를 사용하기 위해 가져온다.
    private Electronic220V electronic220V;

		// 3. 생성자통해 초기화 시켜준다.
    public SocketAdapter(Electronic220V electronic220V){
        this.electronic220V = electronic220V;
    }

    @Override
    public void powerOn() {
				// 4. 이렇게 220v 안의 connect를 사용할 수 있게된다(변환 완료)
        electronic220V.connect();
    }
}
```

이렇게 변환해줄 어뎁터 클래스를 구현하고 메인으로 돌아가 아래와 같이 사용하면 잘 호환이 되는 것을 볼 수 있다.

- 호환된 청소기 예제
```java
public static void main(String[] args) {
        HairDryer hairDryer = new HairDryer();
				connect(hairDryer);

				// 호환되는 청소기
        Cleaner cleaner = new Cleaner();
				Electronic110V adapter = new SocketAdapter(cleaner);
				connect(adapter);
    }
```
