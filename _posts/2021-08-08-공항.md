### 문제

![](https://jihun-24k.github.io/assets/images/baek_img/airport1.png){: .align-center}

- 공항은 G개, 비행기는 P개가 있다.
- 비행기는 주어진 숫자 이하 게이트 중 하나에 도킹이 가능
- 만일 도킹할 수 없는 비행기가 생기면 공항이 폐쇄
- 비행기를 가장 많이 도킹할 수 있는 수를 출력
    
---

### 문제 접근 방법

- 그리디 알고리즘으로 비행기를 가능한 주어진 숫자부터 도킹시키며 최대한 많이 도킹 시켜야한다.
- 각각의 숫자를 이중 for문으로 구하게 되면 시간 초과가 나오기에 union & find 알고리즘(서로소)를 사용한다.
- 초기 각 게이트는 그 숫자만큼 비행기를 도킹 가능
    - 예) 3번 게이트 이하는 최대 3대까지 가능
- 도킹에 완료했으면 결과값을 +1

---

### 풀이 코드

```python
# 백준 10775번 공항
import sys
sys.setrecursionlimit(int(1e6))
input = sys.stdin.readline

# union find 이론
# 게이트 찾기
def find_gate(gates, plane):
    if gates[plane] != plane:
        gates[plane] = find_gate(gates, gates[plane])
    return gates[plane]
# 합치기
def union_gate(gates, a, b):
    a = find_gate(gates, a)
    b = find_gate(gates, b)
    if a > b:
        gates[a] = b
    else:
        gates[b] = a

G = int(input())
P = int(input())

result = 0
# 게이트 초기화
gates = [0] * (G + 1)
for i in range(1, G + 1):
    gates[i] = i

# 비행기 입력
plane = [0] * (P + 1)
for i in range(1, P + 1):
    plane[i] = int(input())

# 연산 시작
for i in range(1, P + 1):
    docking = find_gate(gates, plane[i])
    if docking != 0:
        union_gate(gates, docking, docking - 1)
        result += 1
    else:
        break
print(result)
```
