## JSON이란?

![](https://jihun-24k.github.io/assets/images/spring/json.png){: .align-center}

JavaScript Object Notation으로 자바스크립트에서 파생된 데이터 표기법입니다. 자바스크립트에서 객체를 만들때 사용됐지만 프로그래밍 언어에 독립적으로 사용이 가능해 다양한 언어에서 사용이 가능합니다.

JSON는 아래와 같이 설명될 수 있습니다.

1. JSON 데이터는 key와 value로 이루어집니다.
2. JSON 데이터는 쉼표( , )로 구분합니다.
3. 객체(object)는 중괄호({}) 로 감싸줍니다.
4. 배열(array)는 대괄호([])로 감싸줍니다.

## JSON 문법

```json
// JSON 데이터 종류들
string  : value
number  : value
boolean : value
object  : value { }
array   : value [ ]
```

### 예1) 유저 목록

```json
{
	"phone_number" : "010-1111-2222",
	"age" : 10,
	"isAgree" : false,
	"account" : {
			"email" : "steve@gmail.com",
			"password" : "1234"
	}
}
```

- `key : value` 형태로 작성한다.
- String은 “”로 감싸주고 int형이나 boolean은 그대로 작성한다.

### 예2) user 조회하는 경우

```json
{
	"user_list" : [
		{
			"account" : "abcd",
			"password" : "1234"
		},
		{
			"account" : "adkdkgn",
			"password" : "1111"
		}
	]
}
```

- 배열은 같은 `key와 value` 의 집합들만 가능하다.
