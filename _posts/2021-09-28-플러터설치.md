### 플러터 설치 과정

이번에 플러터 스터디를 진행하게 되면서 플러터 개발환경을 갖춰야했다. 워낙 구글에서 지원을 잘해놓아서 공식 페이지에 들어가 설치 홈페이지를 보면 윈도우, 맥, 리눅스등 각 OS 별로 설치과정을 잘 설명돼있었다. 이 게시물에선 맥OS 환경에서 설치하는 과정을 설명해볼 생각이다. 플러터를 시작하기 전에 아래 세 가지를 설정해야한다. 구글에서 추천하는 IDE가 안드로이드 스튜디오라 IDE는 안드로이드 스튜디오로 진행하기로 했다.

1. 플러터SDK: 플러터 개발을 위함 개발자 킷
2. 안드로이드 스튜디오: 안드로이드 개발을 위한 IDE
3. 안드로이드SDK: 안드로이드 개발을 위한 SDK

### 플러터 SDK 다운로드

압축된 flutter sdk를 원하는 위치에 압축을 풀어라고 하지만 필자는 바로 Downloads에 압축이 풀린 상태로 저장이 돼 다른 파일에 옮겼다.

```bash
$ cd ~/development
$ unzip ~/Downloads/flutter_macos_vX.X.X-stable.zip
```

플러터 도구를 환경변수 path에다가 추가한다.

```bash
$ export PATH="$PATH:`pwd`/flutter/bin"
```

그 다음으론 공식 홈페이지에서 선택적으로 플러터 개별 바이너리를 사전 다운로드해라고 했는데
