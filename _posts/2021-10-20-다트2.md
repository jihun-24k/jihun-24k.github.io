### 다트 함수

함수는 어떤 입력을 넣어 원하는 결과를 얻는 것이라고 생각하면 됩니다. 기본적으로 C언어랑 같지만 다트만의 새로운 유형의 함수들도 존재합니다. 여기선 새로운 기능만을 다룰 예정입니다.

```dart
void main(){
  // 선택 인자(positional optional parameter)
  int num1 = 100;
  int num2 = addNumber(100);      //inc = 1 대입
  int num3 = addNumber(100, 20);  //inc = 20 대입
  print(num1);
  print(num2);
  print(num3);

  // 이름 있는 인자(named optional parameter)
  String http1 = getHttp('http://naver.com', port: 80);
  String http2 = getHttp('http://localhost');
  print(http1);
  print(http2);
}

String getHttp(String url, {int port = 8080}){
  return 'get http from $url , port = $port';
}

int addNumber(int num, [int inc = 1]) => num + inc;
```

#### 선택인자(positional optional parameter)

다트가 선택인자를 제공하는 방법은 두가지 인데 그 중 하나가 선택인자라는 함수입니다. 예를 들어 위 예제 코드인 addNumber() 함수는 두가지 인자를 필요로 합니다. num인자는 필수이고 inc인자는 선택입니다. 만약 inc인자를 넣지 않으면 기본값인 1이 자동으로 대입됩니다.

#### 이름 있는 인자(named optional parameter)

선택인자에민 이름을 넣을 수 있는 함수입니다. 예를들어 getHttp() 함수는 필수적인 url인자와 선택적인 port인자를 갖고 있습니다. port인자를 입력할 때 이름을 붙여 <b>port:80</b>과 같이 맵 형태로 넣습니다.

### 다트의 객체와 클래스

다트의 모든 것은 객체(object)로 이루어져있습니다. 

```dart
void main(){
  // Game 클래스 생성
  Game game1 = Game('Star Craft', 'Strategy');
  Game game2 = ArcadeGame('Strike 1945', 'Shooting', true);

  // 속성 (getter, setter)
  print('game1 is ${game1.name}');
  print('game2 is ${game2.name}');

  game1.genre = 'Realtime Strategy';

  // 메서드 호출
  game1.play();
  game2.play();
}

class Game { // 부모 클래스
  late String _name;
  late String _genre;

  Game(this._name, this._genre); // 생성자

  //getter, setter
  String get name => _name;
  set genre(g) => this._genre = g;

  void play(){
    print('play $_name game($_genre)!!');
  }
}

class ArcadeGame extends Game{ // 자식 클래스
  bool _joystickSupport = false;
  
  // 생성자
  ArcadeGame(String name, String genre, this._joystickSupport) : super(name, genre);
  
  @override // 메서드 오버라이드
  void play(){
    print('$name supports joystick? $_joystickSupport');
  }
}
```

- Game 클래스는 내부에 _name과 _genre라는 멤버 변수를 갖고 있으며 클래스를 생성하기 위해선 두 값을 넣어야합니다. _name은 name 속성을 통해 값을 얻을 수 있고 genre 속성을 사용하면 객체가 생성된 후에도 값을 수정할 수 있습니다.

- ArcadeGame은 Game의 자식 클래스로 Game 클래스의 모든 속성과 메서드를 그대로 사용 가능합니다. @override인 함수는 재정의한 함수입니다.

- 클래스를 사용하기 위해 객체를 만듭니다. 대부분의 생성자 역할은 멤버변수를 초기화할 때 쓰입니다. <b>super(name, genre)</b>는 부모 클래스의 생성자를 먼저 호출한다는 의미입니다.

- 생성된 객체는 인스턴스라 불립니다.

### 다트의 기본 자료구조

프로그램의 성능 효율성을 높이기 위해 사용되는 것이 자료구조입니다. 다트의 대표적인 자료구조는 리스트(List), 집합(Set), 맵(Map)이 있습니다.

1. 리스트(List)
- 같은 데이터 타입을 여러 개 저장할 수 있는 자료구조
```dart
List<int> numbers = [100, 200, 300];
List<String> planets = ['Earth', 'Jupiter'];
```
2. 집합(Set)
- 동일한 값을 허용하지 않는 자료구조
- 리스트에 지원되는 +는 집합에서는 제공되지 않음
- union(), intersection(), difference()메서드가 있음
```dart
Set<int> naturalNumbers = {1, 2, 3, 4, 1};
// 출력값은 1,2,3,4만 나옴
```
3. 맵(Map)
- 키(key)와 값(value)로 이루어져 있음
```dart
Map<int, String> intMap = {
    0: 'AAA',
    50: 'BBB',
    100: 'CCC'
};
```
