### 성능 향상과 메모리 잘 쓰는 법

- 대부분 언어들에서 메모리 초과가 뜨는 이유는 메모리 재할당이 일어나기 때문이다.
- 주로 for 문에서 일어난다.
- 리스트에 append를 하면서 메모리 재할당이 된다.

```python
import time

start_time = time.time()
a = list(range(100000))
a2 = list()

for i in a:
    a2.append(i*2)

end_time = time.time() 
fin = end_time - start_time
print(fin)
#결과
0.04319477081298828
```

### comprehension 방법

```python
import time

start_time = time.time()
temp = [x*2 for x in range(100000)]
end_time = time.time()
fin = end_time - start_time
print(fin)
#결과
0.01129007339477539
```

---

### map과 lamda를 이용한 방법

```python
import time

start_time = time.time()
a = list(range(100000))
a2 = map(lambda n: n*2, a)
end_time = time.time()

fin = end_time - start_time
print(fin)
#결과
0.0034818649291992188
```
