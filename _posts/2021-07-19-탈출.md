### 문제

![](https://jihun-24k.github.io/assets/images/baek_img/escape.png){: .align-center}

- S, D는 시작 위치와 도착 위치
- '*', 'X', '.' 은 물, 돌, 빈 공간
- 매 분마다 물은 불어나고 고슴도치는 한칸씩 이동
- 결과값은 도착할 수 있는 최단시간
    - 만일 도착할 수 없으면 -1을 리턴

    
---

### 문제 접근 방법

- 방향을 이용한 BFS로 풀이 접근
- 고슴도치는 같은 시간 물이랑 있을 수 없다.
    - 그래서 큐에 물부터 먼저 넣어 같은 시간에 물이 먼저 차게한다.
- 큐에는 위치 정보와 물인지 고슴도치인지 구분하는 변수, 경과 시간을 넣는다.

---

### 풀이 코드

```python
# 백준 3056번 탈출
from collections import deque

dx = [1, -1, 0, 0]
dy = [0, 0, 1, -1]

def bfs():
    global result
    q = deque()

    for r in range(R):
        for c in range(C):
            if forest[r][c] == 'S':
                q.append((0, r, c, 'S'))
                visit[r][c] = True
            elif forest[r][c] == '*':
                q.appendleft((0, r, c, '*'))

    while q:
        # d는 걸린 분, xy는 위치, z는 물 또는 고슴도치
        d, x, y, z  = q.popleft()

        for i in range(4):
            nx = x + dx[i]
            ny = y + dy[i]

            if nx < 0 or nx >= R or ny < 0 or ny >= C:
                continue

            # 만약 고슴도치가 동굴을 무사히 갔으면
            if forest[nx][ny] == 'D' and z == 'S':
                result = d + 1
                break
            
            if forest[nx][ny] == '.':
                if z == '*':
                    forest[nx][ny] = '*'
                    q.append((d + 1, nx, ny , z))
                elif not visit[nx][ny]:
                    visit[nx][ny] = True
                    q.append((d + 1, nx, ny, z))
        if result != 0:
            break

R, C = map(int, input().split())
result = 0
forest = [list(input()) for _ in range(R)]
visit = [[False for _ in range(C)] for _ in range(R)]

bfs()
if result == 0:
    print('KAKTUS')
else:
    print(result)
```

### 문제 후기

 처음 접근했을땐 고슴도치가 갔던 자리를 방문체크를 안해서 계속 큐에 append를 수행해 메모리 초과가 뜬거 같았다. 고친 다음 에도 메모리 초과가 뜨기에 정답을 찾으면 바로 break로 for문과 while문을 나가서 함수를 종료시키는 방법으로 줄여나갔다. 

이후 틀림에서는 처음에 물이 지나간 자리를 그냥 돌로 바꿔서 고슴도치가 못가게 했는데 물로 하고 큐를 넣어야지 더 진행되는거였다.. 담엔 실수를 줄여봐야할 듯 하다.
